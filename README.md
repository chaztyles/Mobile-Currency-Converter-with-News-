## Key Features

- Rates Dashboard
  - Switch base currency
  - Customize favorite currencies
  - View popular currencies

- Conversion
  - Convert currencies
  - Convenient reverse conversion option

- News
  - Stay informed with regularly updated news	

- Dark Mode

- Local Storage 

- API Integration

  - News API: https://newsapi.org/
